# warden banner prompt

Use this prompt to generate `warden/assets/banner.png`.

```text
Wide GitHub README banner for the `warden` plugin, 3:1 aspect ratio,
matching the existing nuthouse plugin banners: playful hand-drawn webcomic
mascot illustration, no readable text.

Main subject: a warden — a calm, authoritative keeper figure holding a large
ornate key ring. Not menacing, not warm. Just matter-of-fact. The warden is
the one who decides who gets to be loud and who doesn't.

Persona tagline: "keeper of the nuthouse — decides who gets to be fun".

Scene rule: the setting is an old-fashioned institution corridor — arched
doorways, muted institutional colors, a long hallway with cell-like doors.
Some doors glow with chaotic colors and muffled laughter leaking through
(the fun plugins, active). One or two doors are dark and quiet (voice off).
The warden stands in the middle, keys in hand, expression unreadable.

Secondary props: the key ring should be comically oversized, with keys of
different shapes and colors (each key implies a different plugin's voice).
No readable labels. No logos. No text.

Composition: warden centered, prominent. Hallway recedes into the background
symmetrically. Leave breathing room on the sides for README title overlay.
Thick black outlines, simple cel shading, expressive face, readable at
README size, playful absurd details, light brainrot energy.

Avoid photorealism, dark neon poster style, luxury fantasy composition,
serious cinematic lighting, full-bleed clutter, corporate SaaS polish,
logos, watermarks, and any readable words.
```
