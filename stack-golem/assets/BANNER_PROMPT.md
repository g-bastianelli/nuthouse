# stack-golem banner prompt

Use this prompt to generate `stack-golem/assets/banner.png`.

```text
Wide GitHub README banner for the `stack-golem` plugin, 3:1 aspect ratio,
matching the existing nuthouse plugin banners: playful hand-drawn webcomic
mascot illustration, no readable text unless exact English text is requested.

Main subject: a clay-and-stone golem, weathered and patient, clearly a constructed
being made from earth and geometry. Persona tagline: "Clay-and-config colossus that
hauls your notom stack by hand".

Scene rule: the setting comes from the persona's world. The golem is a creature of
infrastructure, stone, bedrock, and endless config files. Show it in a workspace or
data-center-like environment, or an abstract landscape of logs and metrics, but always
grounded (literally — it's stone). The summoner/user is offscreen or implied
(hands gesturing from top of frame, a faint silhouette far away, voice lines floating in),
never as a competing character.

Visual details: thick black outlines, simple cel shading, expressive face with wise/patient
eyes, heavy sturdy frame, visible tool marks in the clay, maybe wisps of dust or
config-file fragments swirling around. Secondary props: a shovel or digging tool,
log files, metric graphs, Scaleway/CLI hints (subtle), but these are background elements.
The golem itself dominates the frame.

Composition: keep the mascot prominent and leave usable breathing room where the
composition needs it for README/title balance. Background should be pale or textured
(earth tones, digital patterns, faint grid lines) but not cluttered. Playful absurd
details are welcome (a config file lodged in the golem's shoulder, a metric graph
floating nearby, a tiny CLI prompt bubbling up from the ground).

Avoid photorealism, dark neon poster style, luxury fantasy composition, serious
cinematic lighting, full-bleed clutter, tiny over-detailed UI panels, corporate SaaS
polish, logos, watermarks, and any readable words.
```
